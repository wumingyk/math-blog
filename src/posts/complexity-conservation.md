# 如何构建一个轻量级个人博客系统

这是在AI帮助下建立的一个博客系统，包括技术方案、实现步骤，到最后能够展示。只需要在遇到问题时，把问题抛给AI就行了。为什么会在这个时间点来弄个博客呢，打算有选择的记录和分享一些个人想法，包括有意思的“面包屑”，还有数学的一些东西。
这个博客没有复杂的框架，本地编辑 Markdown、推送到github仓库，再自动同步到免费的个人博客平台。如果你感兴趣，也可以动手试试。
## 博客特点

* **文章以 Markdown 存储**
* **本地编辑 → 自动更新**
* **无需后端**
* **轻量、易部署**

## 关键思路

1. 提出个人需求给AI,比如不要申请域名、不需要备案，本地编辑推送更新。
2. 选择一个实现方案，我这里选择的是Vite / VitePress 体系同步加载到github。
3. 不懂代码没关系，这种轻量级的博客，AI写起来轻车熟路。
4. 添加了RSS订阅功能。


## 总结

信息文明的时代，选择主动接受的信息范围很重要，注意力毕竟有限。用这样一个平台去积累，同时也可以分享给需要的人。今天是2026年1月1日，希望自己能坚持下来。
